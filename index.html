#   Accordion

ใช้สำหรับซ่อน Content ที่ต้องการ เปิด/ปิด การแสดงผล โดยการใส่ class .bio-accordion บน container ที่ต้องการ และใส่ Content ด้านใน container ที่ใช้ class .bio-accordion-content

คุณสามารถใช้ร่วมกับ input ได้สองแบบคือ checkbok (เปิดหลาย accordion ได้พร้อมกัน) และ radio (เปิดทีละ accordion) โดยใช้ Label ของ input เป็นตัวเปิดปิดตามสถานะ ::checked ของ input

Markup

<!-- Accordion Markup -->
<div class="bio-accordion">
  <input id="accordion" type="checkbox">
  <label for="accordion">Accordion</label>
  <div class="bio-accordion-content">
    // Content goes here...
  </div>
</div>

Component Options

Class Options
.bio-accordion -has-icon
.bio-accordion-content -

#   Breadcrumb

ใช้สำหรับแสดงโครงสร้างการ Navigation ในหน้าปัจจุบัน โดยการใส่ class .bio-breadcrumb บน container จะส่งผลให้ direct children ทุก node ด้านใน container นี้ จะกลายเป็น list ของ Breadcrumb

คุณสามารถใช้งาน class .bio-breadcrumb บน element ใดๆ ก็ได้ (เช่น div) แต่เราแนะนำให้คุณ ใช้ class นี้บน ul และ ใช้ li เป็น list ของ breadcrumb เพื่อความเป็นระเบียบ และง่ายต่อการอ่าน

Markup

<!-- Breadcrumb Markup -->
<ul class="bio-breadcrumb">
  <li>Home</li>
  <li>item</li>
</ul>
  • Home
  • Item

คุณสามารถใช้งานร่วมกับ <a> หรือ class .bio-link ด้านใน list เพื่อทำเป็น link ที่สวยงาน หรือใช้งาน font icon อื่นๆ ร่วมด้วยก็ได้ เช่น Material Icon เป็นต้น

ตัวอย่างการใช้งานหลายๆ รูปแบบ

<!-- Breadcrumb Markup -->
<ul class="bio-breadcrumb">
  <li><a class="bio-link" href="#">Home</a></li>
  <li><a class="bio-link -primary" href="#">Category</a></li>
  <li><a class="bio-link -primary -fancy" href="#">Category</a></li>

  <li><a class="bio-link -warning" href="#">Category</a></li>
  <li><a class="bio-link -warning -fancy" href="#">Category</a></li>
  <li><a class="bio-link -info" href="#">Category</a></li>
  <li><a class="bio-link -info -fancy" href="#">Category</a></li>
  <li>item</li>
</ul>

#   Button

คุณสามารถใช้ class .bio-button ลงบน element ใดๆ เพื่อให้แสดงผลเป็น button โดยคุณสามารถใช้คู่กับ options ต่าง ของ bio-button ได้เช่น color, style และ state ดังแสดงในตาราง

เพื่อความเหมาะสม เราแนะนำให้คุณใช้ .bio-button บน element <div> หรือ <input type="submit">

Markup

<!-- Button Markup -->
<input type="submit" class="bio-button" value="Click to submit">
<div class="bio-button">Button</div>
Button

Component Options

Class Options
.bio-button

colors: -primary, -accent, -positive, -negative, -warning, -info, -white, -gray, -dark, -black

states: -loading, -disabled

styles: -outline
