#   Breadcrumb

ใช้สำหรับแสดงโครงสร้างการ Navigation ในหน้าปัจจุบัน โดยการใส่ Class .bio-breadcrumb บน Element ที่ต้องการจะส่งผลให้ Direct Children ทุก Node ด้านใน Element นี้ จะกลายเป็น List ของ Breadcrumb

คุณสามารถใช้งาน Class .bio-breadcrumb บน Element ใดๆ ก็ได้ (เช่น <div>) แต่เราแนะนำให้คุณ ใช้ Class นี้บน <ul> และ ใช้ <li> เป็น List ของ Breadcrumb เพื่อความเหมาะสมและเป็นระเบียบ

Markup

<!-- Breadcrumb Markup -->
<ul class="bio-breadcrumb">
  <li>Home</li>
  <li>item</li>
</ul>
  • Home
  • Item

คุณสามารถใช้งานร่วมกับ <a> หรือ Class .bio-link ด้านใน List เพื่อทำเป็น Link ที่สวยงาน หรือใช้งาน Font Icon อื่นๆ ร่วมด้วยก็ได้ เช่น Material Icon เป็นต้น

ตัวอย่างการใช้งานหลายๆ รูปแบบ

<!-- Breadcrumb Markup -->
<ul class="bio-breadcrumb">
  <li><a class="bio-link" href="#">Home</a></li>
  <li><a class="bio-link -primary" href="#">Category</a></li>
  <li><a class="bio-link -primary -fancy" href="#">Category</a></li>
  <li><a class="bio-link -positive" href="#">Category</a></li>
  <li><a class="bio-link -positive -fancy" href="#">Category</a></li>
  <li><a class="bio-link -negative" href="#">Category</a></li>
  <li><a class="bio-link -negative -fancy" href="#"><i class="material-icons _fs-5">local_shipping</i> Category</a></li>
  <li><a class="bio-link -warning" href="#">Category</a></li>
  <li><a class="bio-link -warning -fancy" href="#">Category</a></li>
  <li><a class="bio-link -info" href="#">Category</a></li>
  <li><a class="bio-link -info -fancy" href="#">Category</a></li>
  <li>item</li>
</ul>
