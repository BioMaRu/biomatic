#   Progress

Progress เหมาะสำหรับแสดงผลให้ผู้ใช้รู้ว่า เขาดำเนินการมาไกลแค่ไหน และอีกไกลแค่ไหนถึงจะเข้าใกล้เป้าหมาย

การใช้งาน Progress ทำได้โดยใส่ Class .bio-progress ลงบน Element <progress>

Markup

<!-- Progress Markup -->
<progress class="bio-progress" value="25" max="100"></progress>

Component Options

Class Options
.bio-progress

Colors: -primary, -accent, -positive, -negative, -warning, -info, -dark, -black

ตัวอย่างการใช้งานอื่นๆ

<!-- Progress Markup -->
<progress class="bio-progress -primary" value="10" max="100"></progress>
<progress class="bio-progress -accent" value="20" max="100"></progress>
<progress class="bio-progress -positive" value="30" max="100"></progress>
<progress class="bio-progress -negative" value="40" max="100"></progress>
<progress class="bio-progress -warning" value="50" max="100"></progress>
<progress class="bio-progress -info" value="60" max="100"></progress>
<progress class="bio-progress -dark" value="70" max="100"></progress>
<progress class="bio-progress -black" value="80" max="100"></progress>

ตัวอย่างการใช้งานอื่นๆ

ปรับขนาดด้วยตั้งค่า CSS Style height: *px หรือใช้งาน Helper Class ._h-* 

<!-- Progress Markup -->
<progress class="bio-progress _h-1px" value="20" max="100"></progress>
<progress class="bio-progress _h-2px" value="20" max="100"></progress>
<progress class="bio-progress _h-4px" value="20" max="100"></progress>
<progress class="bio-progress _h-8px" value="20" max="100"></progress>
<progress class="bio-progress _h-16px" value="20" max="100"></progress>
<progress class="bio-progress _h-32px" value="20" max="100"></progress>
<progress class="bio-progress _h-64px" value="20" max="100"></progress>
