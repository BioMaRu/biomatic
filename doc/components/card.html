#   Card

Card เป็น Content Container ที่มีความยืดหยุ่นสูง สำหรับแสดงผลข้อมูลชุดเดียวกันไว้ด้วยกัน

การใช้งาน Card ทำโดยการใส่ Class .bio-card บน Element ที่ต้องการ

Markup

<!-- Card Markup -->
<div class="bio-card">
  <img class="bio-card-cover" src="...">
  <div class="bio-card-title"> ... </div>
  <div class="bio-card-subtitle"> ... </div>
  <div class="bio-card-content">
    ...
  </div>
  <div class="bio-card-actions">
    ...
  </div>
</div>

Component Options

Class Options
.bio-card styles: -horizontal
.bio-card-cover -
.bio-card-subtitle -
.bio-card-content -
.bio-card-actions styles: -push-left, -attached,

  <img class="bio-card-cover" src="https://placeimg.com/640/480/animals" alt="image">
  <div class="bio-card-title"><strong>Hello Kitty</strong></div>
  <div class="bio-card-subtitle">@BioMaRu</div>

  </div>
</div>
