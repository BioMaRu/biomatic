#   Message

Message เหมาะสำหรับแสดง Content ที่ต้องการดึงความสนใจผู้ใช้และเน้นความรู้สึกต่างๆ เช่นการให้ข้อมูล, การแจ้งเตือน, การแสดงข้อผิดพลาด เป็นต้น

คุณสามารถใช้งาน Message ได้โดยใส่ Class .bio-message บน Element ที่ค้องการ

หากคุณต้องการจุดสำหรับแสดงปุ่มปิด คุณสามารถใส่ child ที่ใช้ Class .bio-message-close ได้ ซึ่งปุ่มปิดจะอยู่มุมขวาบนของ Message เสมอ โดยคุณอาจจะใช้ Utility Class .u-close เพื่อช่วย แสดงปุ่มปิดที่สวยงามได้

Markup

<!-- Message Markup -->
<div class="bio-message _mgbt-8px">
  <div class="bio-message-close">...</div>
  Lorem ipsum dolor sit amet, consectetur adipiscing elit.
</div>

Component Options

Class Options
.bio-message

Colors: -accent, -positive, -negative, -warning, -info, -dark

.bio-message-close -
