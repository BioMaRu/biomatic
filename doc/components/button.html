#   Button

คุณสามารถใช้ class .bio-button ลงบน element ใดๆ เพื่อให้แสดงผลเป็น button โดยคุณสามารถใช้คู่กับ options ต่าง ของ bio-button ได้เช่น color, style และ state ดังแสดงในตาราง

เพื่อความเหมาะสม เราแนะนำให้คุณใช้ .bio-button บน element <div> หรือ <input type="submit">

Markup

<!-- Button Markup -->
  <input type="submit" class="bio-button" value="Click to submit">
  <div class="bio-button">Button</div>
Button

Component Options

Class Options
.bio-button

colors: -primary, -accent, -positive, -negative, -warning, -info, -white, -gray, -dark, -black

states: -loading, -disabled

styles: -outline

#   Card

Card คือ layout container ที่มีความยืดหยุ่นสูง สำหรับแสดงผล content class

ใช้งาน Card โดยการใส่ container class .bio-card บน element ที่ต้องการ

Markup

<!-- Card Markup -->
  <div class="bio-card">
    <img class="bio-card-cover" src="...">
    <div class="bio-card-title"> ... </div>
    <div class="bio-card-subtitle"> ... </div>
    <div class="bio-card-content">
      ...
    </div>
    <div class="bio-card-actions">
      ...
    </div>
  </div>

Component Options

Class Options
.bio-card styles: -horizontal
.bio-card-cover -
.bio-card-subtitle -
.bio-card-content -
.bio-card-actions styles: -push-left, -attached,
