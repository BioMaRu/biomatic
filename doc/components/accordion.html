#   Accordion

เหมาะสำหรับ Content ที่ต้องการ เปิด/ปิด การแสดงผลได้ ใช้งานโดยการใส่ Class .bio-accordion บน Container ที่ต้องการ และใส่ Content ไว้ด้านใน Container ที่ใช้ Class .bio-accordion-content

คุณสามารถใช้ร่วมกับ <input> ได้สองแบบคือ Checkbok (เปิดหลาย Accordion ได้พร้อมกัน) และ Radio (เปิดทีละ Accordion) โดยใช้ <label> ของ <input> เป็นตัวเปิดปิดตามสถานะ ::checked

Markup

<!-- Accordion Markup -->
<div class="bio-accordion">
  <input id="accordion" type="checkbox">
  <label for="accordion">Accordion</label>
  <div class="bio-accordion-content">
    // Content goes here...
  </div>
</div>
