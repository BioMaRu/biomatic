#   Border

Class ที่เกี่ยวข้องกับ Border จะมีสามส่วนคือ กำหนด Border Width, กำหนด Border Color และกำหนด Border Radius

Border Width ใช้ Base ._bd, ._bdl, ._bdr, ._bdt, ._bdbt สำหรับกำหนดขนาด Solid Border ทุกด้าน หรือด้านใดด้านหนึ่ง โดยค่าตั้งต้นจะเป็น Border Style Solid และ Border Color จะเป็น $color-dark

Base Value

._bd, ._bdl, ._bdr, ._bdt, ._bdbt

Pixel Value: 0px 1px 2px 4px 8px 16px 32px

Border Radius

Border Radius ใช้ Base
._bdrd สำหรับ border-radius
._bdtlrd สำหรับ border-top-left-radius
._bdtrrd สำหรับ border-top-right-radius
._bdbtlrd สำหรับ border-bottom-left-radius
._bdbtrrd สำหรับ border-bottom-right-radius
ส่วน Value เป็นหน่วยต่างๆ ดังตาราง แต่สำหรับ Border Radius นั้นจะพิเศษตรงที่มี Value 3px ซึ่งเป็นค่าที่มักถูกใช้บ่อย และค่า max สำหรับทำ pill style ซึ่งกำหนดไว้ที่ 9999px

Base Value

._bdrd, ._bdtlrd, ._bdtrrd, ._bdbtlrd, ._bdbtrrd

Pixel Value: 0px 1px 2px 3px 4px 8px 16px 32px 64px 128px

Percentage Value: 50pct

Other Value: max
