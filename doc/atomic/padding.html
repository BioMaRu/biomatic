#   Padding

Base คือ ._pd สำหรับ padding ทุกด้าน

._pdl, ._pdr, ._pdt, ._pdbt สำหรับ padding left, right, top และ bottom ตามลำดับ

._pdv สำหรับ padding ในแนวตั้ง และ ._pdh สำหรับ padding ในแนวนอน สำหรับ padding แต่ละด้าน

Base Value

._pd ._pdl ._pdr ._pdt ._pdbt ._pdv ._pdh

Pixel Value: 0px 1px 2px 4px 8px 16px 32px 64px 128px 256px

Other Value: auto
