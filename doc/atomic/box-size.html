#   Box Size

Class ทราเกี่ยวข้องกับ Box Size คือ width, height และ max-width

Base Value

._w, ._h, ._mxw

Pixel Value: 0px 1px 2px 4px 16px 32px 64px 128px 256px 512px

Percentage Value: 5pct 10pct 15pct 20pct 25pct 30pct 35pct 40pct 45pct 55pct 60pct 65pct 70pct 75pct 80pct 85pct 90pct 95pct 100pct

Other Value: auto
