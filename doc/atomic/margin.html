#   Margin

Base คือ ._mg สำหรับ margin ทุกด้าน

._mgl, ._mgr, ._mgt, ._mgbt สำหรับ margin left, right, top และ bottom ตามลำดับ

._mgv สำหรับ margin ในแนวตั้ง และ ._mgh สำหรับ margin ในแนวนอน สำหรับ margin แต่ละด้าน

Base Value

._mg ._mgl ._mgr ._mgt ._mgbt ._mgv ._mgh

Pixel Value: 0px 1px 2px 4px 8px 16px 32px 64px 128px 256px

Other Value: auto
