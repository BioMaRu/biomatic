#   Color

Color คือกลุ่ม Atomic Class ของ Style color และ background-color

Color

Base คือ ._cl สำหรับ color และ ._bgcl สำหรับ background-color โดย Value จะเป็นชื่อสีตามที่กำหนดใน biomatic

สีจะแบ่งออกเป็น 3 กลุ่มคือ Theme Color, Semantic Color และ Grayscale Color

Theme Color จะกลุ่มเป็นสีหลักที่คุณจะสนใจ เพราะเป็นสีที่กำหนดสีหลัก สีเน้น และสีกลาง เป็นตัวทำหนดภาพรวมของสีบนเว็บไซต์ของคุณ ซึ่งคุณสามารถเปลี่ยนสีเหล่านี้ได้ที่ไฟล์ สีทั้งหมดนี้ถูกกำหนดไว้ที่ config.scss

Semantic Color จะกลุ่มเป็นที่ใช้สำหรับแสดงอารมต่างๆ เช่น ปลอกภัย ผิดพลาด อันตราย เป็นต้น

Grayscale Color จะกลุ่มเป็นที่ใช้สำหรับเป็นสีพื้นใน UI ที่ไม่ต้องการความโดดเด่นเช่น สีของตัวอักษรใน Paragraph สีพื้นของ Card เป็นต้น

ตารางรายการ Value ในกลุ่มต่าง

กลุ่มสี Value
Theme Color

primary accent neutral

Semantic Color

positive negative warning info

Grayscale Color

white gray dark black
