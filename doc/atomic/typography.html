#   Typography

Typography คือกลุ่ม Atomic Class ที่เกี่ยวข้องกับ Font และ Text ประกอบไปด้วย font-size, font-weight, line-height, text-align, text-transform, white-space และ letter-spacing

Font Size

Base คือ ._fs และ Value จะเป็นค่าที่ไม่มีหน่วย แสดงถึงลำดับความใหญ่ของตัวอักษรตั้งแต่ค่า 1 ถึง 8

ตารางเปรียบเทียบ

Class Style
._fs-1 font-size: 3rem
._fs-2 font-size: 2.5rem
._fs-3 font-size: 2rem
._fs-4 font-size: 1.5rem
._fs-5 font-size: 1.25rem
._fs-6 font-size: 1rem
._fs-7 font-size: 0.75rem
._fs-8 font-size: 0.5rem

Example

<!-- Markup -->
<p class="_fs-1">Hello World</p>
<p class="_fs-2">Hello World</p>
<p class="_fs-3">Hello World</p>
<p class="_fs-4">Hello World</p>
<p class="_fs-5">Hello World</p>
<p class="_fs-6">Hello World</p>
<p class="_fs-7">Hello World</p>
<p class="_fs-8">Hello World</p>

Text Align

Base คือ ._tal และ Value คือตัวย่อของค่าใน Style text-align

Class Value
._tal

Left: l
Right: r
Center: ct
Justify: jtf

Text Transform

Base คือ ._ttf และ Value คือตัวย่อของค่าใน Style text-transform

Class Value
._ttf

Uppercase: upc
Lowercase: lwc
Capitalize: cptl
None: n

Example

<!-- Markup -->
<p class="_ttf-upc">Meowwloo</p>
<p class="_ttf-lwc">Meowwloo</p>
<p class="_ttf-cptl">Meowwloo</p>
<p class="_ttf-n">Meowwloo</p>

White Space

Base คือ ._wsp และ Value คือตัวย่อของค่าใน Style white-space

Class Value
._wsp

Normal: nm
No wrap: nw
Pre: p
Pre Line: pl
Pre Wrap: pw

Example
