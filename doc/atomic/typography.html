#   Typography

Typography คือกลุ่ม Atomic Class ของ Style font-size, font-weight

Font Size

Base คือ ._fs และ Value จะเป็นค่าที่ไม่มีหน่วย แสดงถึงลำดับความใหญ่ของตัวอักษรตั้งแต่ค่า 1 ถึง 8

ตารางเปรียบเทียบ

Class Style
._fs-1 font-size: 3rem
._fs-2 font-size: 2.5rem
._fs-3 font-size: 2rem
._fs-4 font-size: 1.5rem
._fs-5 font-size: 1.25rem
._fs-6 font-size: 1rem
._fs-7 font-size: 0.75rem
._fs-8 font-size: 0.5rem

Example

<!-- Markup -->
<p class="_fs-1">Hello World</p>
<p class="_fs-2">Hello World</p>
<p class="_fs-3">Hello World</p>
<p class="_fs-4">Hello World</p>
<p class="_fs-5">Hello World</p>
<p class="_fs-6">Hello World</p>
<p class="_fs-7">Hello World</p>
<p class="_fs-8">Hello World</p>

Font Weight

Base คือ ._fw และ Value คือเป็นค่าที่ไม่มีหน่วยตรงตามค่าที่ใช้ใน Style font-weight ตั้งแต่ 100 ถึง 900

Example

<!-- Markup -->
<p class="_fw-100">Hello World</p>
<p class="_fw-200">Hello World</p>
<p class="_fw-300">Hello World</p>
<p class="_fw-400">Hello World</p>
<p class="_fw-500">Hello World</p>
<p class="_fw-600">Hello World</p>
<p class="_fw-700">Hello World</p>
<p class="_fw-800">Hello World</p>
<p class="_fw-900">Hello World</p>
