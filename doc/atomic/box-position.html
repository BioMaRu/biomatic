#   Position

Class มีสองส่วนคือ กำหนดรูปแบบ position และกำหนดค่าของ Position ทิศต่างๆ (Left, Rignt, Top และ Bottom) ของ Element

สำหรับ Position Base คือ ._pst และ Value คือ ตัวย่อของชื่อ Position

สำหรับค่า Position ของด้านต่างๆ Base คือ ._t ._bt ._l ._r และ Value คือ ค่าในหน่วยต่างๆ

Base Value

._pst

static: stt
relative: rlt
absolute: asl
fixed: f

._l, ._r, ._t, ._bt

Pixel Value: 0px 1px 2px 4px 8px 16px 32px 64px 128px 256px

Percentage Value: 50pct

Other Value: auto

Position
